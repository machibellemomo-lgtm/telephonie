Théorie du Trafic Téléphonique

Le trafic téléphonique est mesuré en Erlang (E) : A = λ × h, où λ est le taux d'arrivée des appels et h la durée moyenne. Agner Krarup Erlang (1878–1929) a établi les modèles mathématiques fondamentaux du dimensionnement des réseaux téléphoniques.

A
Trafic Erlang
A = λ × h — Intensité du trafic offert au système de commutation.
B
Erlang B (Perte)
Sans file d'attente (M/M/N/N). Appels bloqués définitivement perdus.
C
Erlang C (Attente)
Avec file d'attente infinie (M/M/N/∞). Appels en attente d'un circuit libre.
FORMULE ERLANG B — PERTES
B(N,A) = (AN/N!) / [Σk=0→N Ak/k!]
Récursif (stable) : B(0,A)=1    B(N,A) = (A/N·B(N-1,A)) / (1 + A/N·B(N-1,A))
FORMULE ERLANG C — ATTENTE
C(N,A) = B(N,A) / [1 − (A/N)·(1 − B(N,A))]    valide pour N > A

Algorithme de Calcul

Saisie des paramètres — Trafic A (Erlang), probabilité cible B_cible (%), modèle (B ou C), nombre de simulations N_sim.
Initialisation — Poser N = ceil(A). Point de départ = charge théorique maximale 100%.
Calcul récursif Erlang — Calculer P(N,A) par formule récursive (évite les dépassements de factorielles). Incrémenter N jusqu'à P ≤ B_cible.
Simulation Monte-Carlo — Générer N_sim arrivées d'appels (loi de Poisson). Affecter les circuits. Calculer P_sim = rejets / N_sim.
Comparaison — Calculer l'écart |ΔP| = |P_erlang − P_sim| et générer le tableau 5 scénarios.
Sortie — Afficher N recommandé, probabilité effective, charge réseau A/N, graphique et tableau comparatif.
